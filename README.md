### 微信小程序－微票

### 说明：

实现微票当前上映电影和影院展示。

### 数据接口:

- https://api.getweapp.com/vendor/bmbstack/cinemaList
- https://api.getweapp.com/vendor/bmbstack/playingList
- https://api.getweapp.com/vendor/bmbstack/bannerList

### 目录结构：

- image — 存放项目图片文件
- style — 存放公共样式文件
- pages — 存放项目页面渲染相关文件
- util — 存放文章数据

### 开发环境：

微信web开发者工具 v0.11.112301
